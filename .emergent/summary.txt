-   **Equipment Hierarchy Manager:** A dedicated page () allows users to manage a full equipment hierarchy.
    -   **ISO 14224 Aligned:** The hierarchy structure (Installation, Plant/Unit, Section/System, Equipment Unit, Subunit, Maintainable Item) and terminology are aligned with the ISO 14224 standard.
    -   **Hierarchy Management:**
        -   Users can create/edit/delete nodes.

        -   A click-based Move Mode allows repositioning nodes between different parents.
        -   Buttons are present to **Promote** (move up a level), **Demote** (move down a level), and **Reorder** (move up/down among siblings).
    -   **Data Integrity:** Backend validation prevents creating nodes with duplicate names under the same parent.
-   **Chatbot Integration:** The chat interface is now tightly integrated with the hierarchy.
    -   It validates that a reported asset exists in the hierarchy before creating a threat.
    -   It can resolve partial tag names (e.g., P-101) to the full equipment name (Pump P-101).
    -   It incorporates the equipment's assigned criticality (e.g., Safety Critical) into the threat's risk score calculation.
-   **Data Libraries:**
    -   The system includes an editable library of equipment types.
    -   A comprehensive Failure Modes Library () is available, containing 138 modes, including a detailed breakdown for Extruders.
-   **Threats Page Integration:** Clicking on an asset in the hierarchy sidebar (visible on the threats page) filters the list to show threats for only that asset.
